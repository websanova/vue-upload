# Vue Upload

A simple, light weight and intuitive upload control for Vue.js.

**NOTE:** Supports Vue 2 and Vue 3. Check [Startup Guide](https://websanova.com/docs/vue-upload/guides/startup#vue-3) for more details on usage.
